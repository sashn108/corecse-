# CoreCSE — Every Concept. Every Year. One Place.

GATE CSE preparation platform targeting AIR 1 (IIT Bombay CSE, IISc Bangalore).

## Features
- Subject-wise concept trees with expandable nodes
- Mastery-level theory notes with GATE-specific traps highlighted
- PYQ bank with year/marks filtering and instant answers
- AI-powered 2027 question predictions based on 20-year pattern analysis
- Topic readiness tracker

## Subjects covered
OS · Data Structures · Algorithms · DBMS · Computer Networks · TOC · Digital Logic · Discrete Maths · CO & Architecture · Compiler Design

## Contributing
Each subject page follows the same template (concept tree + theory + PYQs + predictions).
To add a new topic, copy `subjects/os.html` and update the content.
