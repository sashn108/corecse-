📐 Key Formulas

EAT = h×t_TLB + (1-h)×(t_TLB + t_mem)

EAT (2-level) = α·t + (1-α)·2t
t = memory access time
Avg Waiting Time = Σ(finish - arrival - burst) / n
Turnaround = Completion - Arrival
